# Changelog

All notable changes to this project will be documented in this file.

## [1.3.0] - 2025-01-08

### Added
- **Mulky OS**: A fully functional web-based operating system experience with an interactive terminal, system monitor, and file browser.
- **AI Kernel**: Integrated `AICompanion` as the core intelligence of Mulky OS with advanced reasoning capabilities.
- **Stripe Payments**: Integrated Stripe for secure access to premium content (Resume).
- **GitHub Integration**: Real-time project fetching via GitHub API.
- **Visual Edits**: Enabled Turbopack-based visual editing for rapid UI iterations.

### Fixed
- Date consistency across documentation.

## [1.2.0] - 2025-01-08

### Added
- **AI Pet (AIPet.tsx)**: A cute, glassmorphic companion that moves randomly across the screen, has emotions, and greets visitors with random speech bubbles.
- **Visitor Tracking**: Integrated visitor tracking functionality.
- **Sound Effects**: Added interaction sound effects (click/hover) throughout the site.

### Optimized
- **SEO & Metadata**: Enhanced search engine optimization with comprehensive meta tags, JSON-LD structured data, and hidden keywords for better discoverability.
- **Performance**: Optimized animations and glassmorphic effects for better rendering performance.

## [1.1.0] - 2025-01-07

### Added
- **Multi-language Support**: Integrated i18n support for Indonesian and English.
- **Glassmorphic UI**: Implemented a modern glassmorphism design system across all components.
- **Thinking Page**: Added a space for sharing thoughts and insights.
- **AI Integration**: Added AI-powered features for project analysis and chat.

## [1.0.0] - 2025-01-01

### Initial Release
- Personal portfolio with Projects, About, and Contact sections.
- Responsive design using Tailwind CSS and Framer Motion.
